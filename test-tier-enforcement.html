🎯 Test Instructions

Open http://localhost:5174 and test these scenarios:

  1. Free Tier Test: Use DevTierSwitcher to set to "Free", then try:
    • Click the 🎤 Mic button → Should show upgrade prompt
    • Click the + button → Menu should appear above the button
    • Try 📎 Add Photo → Should show upgrade prompt
    • Try 📷 Take Photo → Should show upgrade prompt
    • Try 🎤 Record Audio → Should show upgrade prompt
  2. Core/Studio Tier Test: Use DevTierSwitcher to set to "Core" or "Studio", then try:
    • Click the 🎤 Mic button → Should work (show "coming soon" message)
    • Click the + button → Menu should appear above the button
    • Try all menu options → Should work (show "coming soon" messages)
    • Click outside the menu → Menu should disappear
    • Press Escape key → Menu should disappear

✅ Expected Results

Free Tier:
  • Mic button shows: "Voice features are available in Core & Studio plans. Upgrade to unlock!"
  • Menu items show: "[feature] features are available in Core & Studio plans. Upgrade to unlock!"
  • Upgrade modal appears when features are blocked
Core/Studio Tier:
  • All features work and show "feature coming soon" messages
  • No upgrade prompts appear
Popper.js Positioning & UX:
  • Attachment menu always appears above the + button
  • Menu stays properly positioned on scroll/resize
  • Menu has smooth animations
  • Clicking outside the menu closes it
  • Pressing Escape key closes the menu

🐛 Troubleshooting

If something isn't working:

  • Check browser console for errors
  • Verify DevTierSwitcher is working (should see tier changes in console)
  • Ensure backend is running on port 3000
  • Check that Popper.js is loaded (should see no positioning errors)
